<h1 align="center">Welcome to TP on CI and CircleCI 👋</h1>


You cand find the codelabs here on iCampus.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
yarn start

# watch mode
yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```

## Lints

```bash
# Check
$ yarn format:check

# Format code
$ yarn format:check

$ yarn lint
```

## Generate

```bash
$ yarn generate module with-crud --name myModule
```
